# Among Us Note

Among Us をやっている途中でスマホでサクサクメモを取るアプリ

## Getting Started

```
$ npm i
$ vercel dev
```

## Deployment

[Vercel](https://vercel.com/)でホスティングしています。

develop ブランチに push されると自動でビルドが走り、仮の URL が発行されます。

master ブランチに push されると自動で https://aunote.site にデプロイされます。
